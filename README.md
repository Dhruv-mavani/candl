# Candl

**Candl is a decentralized market protocol on Solana that enables continuous price discovery and instant liquidity for unique digital assets using bonding curves.**

Instead of listing an NFT and waiting for a buyer, Candl turns every NFT into its own automated market maker (AMM). 

Every asset gets:
- Instant buy and sell execution.
- Live candlestick charts.
- Transparent reserve backing.
- Continuous price discovery.
